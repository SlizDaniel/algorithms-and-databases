3. **cyclic list implementation**

5. **binary search tree implementation**

8. **graph implementation**
9. **Ford-Fulkerson algorithm**
10. **minimum spanning tree/Prim algorithm**
11. **Ullmann algorithm**
12. **pattern search algorithms**
   - bloom filter implementation
13. **approximate string matching using dynamic programming**

14. **convex hull finding algorithm**
